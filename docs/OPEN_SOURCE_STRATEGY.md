# SkillClip Open Source Strategy

## Goal

Position SkillClip as the open-source default for turning AI chats into reusable skills.

The breakout angle is not:

- another prompt sidebar
- another snippet manager
- another multi-model chat wrapper

The breakout angle is:

`Turn AI chats into reusable skills.`

## Core Narrative

Use the same framing everywhere:

- Not a prompt manager
- An AI skill compiler
- Local-first and exportable
- Works across AI products

## What Makes It Shareable

- obvious pain point
- clear before and after demo
- small but magical shortcut-based UX
- Git-friendly assets people can inspect and fork
- category-defining language

## Launch Sequence

### Phase 1: Build in Public

- publish landing README early
- share product thesis and data model screenshots
- show `conversation -> skill draft -> reusable skill` flow

### Phase 2: Demo the Magic

- short demo video of capture and slash reuse
- side-by-side comparison with normal prompt managers
- examples like PRD writing, SQL debugging, Xiaohongshu content, and support reply drafting

### Phase 3: Open Source Credibility

- keep skill files human-readable
- expose adapter system for contributors
- document export format and roadmap clearly

### Phase 4: Community Pull

- accept community-contributed platform adapters
- accept public skill examples
- highlight forks and variants as first-class contributions

## Best Early GitHub Assets

- polished README
- architecture diagram
- animated GIF demo
- sample skills folder
- contributor-friendly roadmap

## Best Content Angles

- “Prompt managers save text. SkillClip saves methods.”
- “Your best AI workflows are buried in old chats.”
- “Compile your AI chats into reusable skills.”
- “Local-first skill system for ChatGPT, Claude, Gemini, and more.”

## Feature Release Order

1. Capture and compile
2. Slash palette reuse
3. Variants and diff
4. Export and Git mode
5. Team sharing

## Community Hooks

- platform adapter contribution guide
- sample skill packs
- public benchmark of reusable AI workflows
- template prompts for common jobs

## Avoid

- overbuilding cloud features too early
- vague “AI productivity” positioning
- becoming a generic prompt database

## Success Signals

- users share compiled skill screenshots, not just saved prompts
- contributors add adapters for new AI tools
- exported skill files get forked and remixed
- community starts using “skill compiler” language
