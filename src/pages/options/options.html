使用说明

如何使用

1. 原始素材：你保存的 prompt 或整段对话原文。

2. 可复用技能：系统会自动整理，你可以直接编辑，或者直接升级为正式技能。

3. 同场景优化版本：同一个技能的第二版、第三版，由你手动创建和修改。

4. 已归档：隐藏出主界面，但仍保留本地，可恢复。

调试说明

1. 如果按钮没有反应，先重新加载扩展，再刷新 AI 页面。

2. 如果没有抓到内容，先确认当前页面属于支持的平台。

3. 如果语音失败，检查 Chrome 的麦克风权限和浏览器支持情况。

4. 如果扩展报错，可在 `chrome://extensions` 里打开 Service Worker 查看运行错误。

校验模式

归档表示从工作台主界面隐藏，但仍保留在本地。删除表示永久移除。

系统会在生成技能、保存编辑、升级为可复用技能时自动运行校验。

只要你配置了模型，Prompt 优化和 Skill 编译就会默认优先走模型；校验仍然可以继续用本地规则或 API 复核。

上线步骤

1. 补最终图标和截图。

2. 发布公开可访问的隐私政策页面。

3. 在 Chrome 里逐个测试支持的平台。

4. 打包扩展根目录，生成发布压缩包。

5. 按隐私披露要求提交到 Chrome Web Store。

知识库联动

你现在可以在保存后自动导出成 Markdown 或 JSON。

当前支持：本地下载、Obsidian、Notion。用户可以自己选择导出目标。

开启后，保存素材或升级技能时会按你的设置自动落地到对应知识库。
