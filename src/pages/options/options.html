Guides / 使用说明

How To Use / 如何使用

1. 原始素材：你保存的 prompt 或整段对话原文。

2. 可复用技能：系统会自动整理，你可以直接编辑，或者直接升级为正式技能。

3. 同场景优化版本：同一个技能的第二版、第三版，由你手动创建和修改。

4. 已归档：隐藏出主界面，但仍保留本地，可恢复。

Debug Checklist / 调试说明

1. If buttons do nothing, refresh the AI page after reloading the extension.

2. If capture is empty, make sure the page is one of the supported AI domains.

3. If voice fails, check Chrome speech permissions and browser support.

4. Open the service worker console from `chrome://extensions` for runtime errors.

Validation Mode / 校验模式

Archive means hide from the main workspace but keep it locally. Delete means remove it permanently.

Validation runs when a pending item is generated, edited, and promoted into a reusable skill.

Model-assisted prompt optimization and skill compilation now use your configured model by default. Validation can still stay local or use API review.

Deployment Path / 上线步骤

1. Add final icons and screenshots.

2. Publish a public privacy policy page.

3. Test each supported AI site in Chrome.

4. Zip the extension root for release packaging.

5. Submit to Chrome Web Store with the privacy disclosures.
