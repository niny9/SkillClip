Workspace

Workspace / 工作台

保存 Prompt 或 Whole Flow 后，会先进入待整理队列。你确认后，它才会变成可复用技能；优化版用于同场景下的第二版、第三版。

Flow / 当前流程

1. 原始素材：你刚保存下来的原文或对话。

2. 可复用技能：原始素材会自动整理，你可以直接编辑，或直接升级成正式技能。

3. 同场景优化版本：你自己从正式技能分出的第二版、第三版。

4. 已归档：不再常用，但仍保留在本地资产里。

Quick Actions / 快捷操作

Open an AI page, then use these buttons to trigger the in-page tools.

`Open Action Bar`：在 AI 页面打开操作条，你可以保存 Prompt、保存对话、语音输入、编译为预览。

`Open Skill Palette`：在 AI 输入框里打开技能插入面板，用来复用已经保存好的正式技能。
