Workspace / 工作台

保存 Prompt 或 Whole Flow 后，会先进入 Inbox，并默认自动生成 Compile Preview。你审核后保存成 Draft，再提升成正式 Skill。

Flow / 当前流程

1. Save Prompt 或 Save Whole Flow：先保存原始素材。

2. 系统会自动生成 Compile Preview / 编译预览。

3. 你确认预览没问题后，点 Save as Draft / 保存为草稿。

4. 草稿确认后，点 Promote to Skill / 升级成正式技能。

5. 如果误归档，可在 Archived / 已归档 里恢复。
