Workspace / 工作台

保存 Prompt 或 Whole Flow 后，会先进入 Inbox，并默认自动生成 Compile Preview。你审核后保存成 Draft，再提升成正式 Skill。

Flow / 当前流程

1. Save Prompt 或 Save Whole Flow：先保存原始素材。

2. 系统会自动生成一个处理中技能。刚生成时叫 Preview，你确认后就变成 Draft。

3. Draft 再点 Promote to Skill / 升级成正式技能，就进入可复用技能区。

4. Variant / 优化版本 是你手动为同一场景分叉出的另一版，不是系统自动猜出来的。

5. 如果误归档，可在 Archived / 已归档 里恢复。

Quick Actions / 快捷操作

Open an AI page, then use these buttons to trigger the in-page tools.

`Open Action Bar`：在 AI 页面打开操作条，你可以保存 Prompt、保存对话、语音输入、编译为预览。

`Open Skill Palette`：在 AI 输入框里打开技能插入面板，用来复用已经保存好的正式技能。
